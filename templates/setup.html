Installing CipherChat

This project is in beta, which means that not all functions are available yet, or that errors may occur.

1. Choose your OS:

Windows macOS Linux
{% else %}
Windows macOS Linux
{% endif %} {% if not os is defined %}
2. Before we start...

We recommend:
- that you only download files via Tor
- verify files before using

Tor can't protect you from everything. Watch out.

3. Download CipherChat:

Now download the ZIP file from one of the following instances:

 GitHub  Tor Hidden Service  Mirror 1  Mirror 2
4. Verify your download:

Select your download...

or verify using the OpenPGP signature.

5. Unzip the file:

If you have not yet installed WinZip, use https://www.winzip.com/en/download/winzip/ to download and install it.

Navigate to the directory in which the "CipherChat.zip" or "CipherChat-master.zip" file is located. This could be your download folder, for example.
Double-click on the file to open it with WinZip. (If another program starts use right click > "Open with..." and select WinZip)
In the WinZip window, select all files by clicking on each checkbox of each file. Click on the "Extract" button in the WinZip interface.
You will then be asked for the destination folder where the extracted files should be saved. Create a new folder called "CipherChat" and go to it, then press "Open".

Or use the following commands:
mkdir CipherChat
expand -r CipherChat.zip -F:* CipherChat

Navigate to the directory that contains the "CipherChat.zip" or "CipherChat-master.zip" file, e.g. the folder in which you downloaded the file.
Double-click on the "CipherChat.zip" file. This should start the built-in "Archive Utility" of macOS. The program will open automatically and begin the extraction process.
The "Archive Utility" will automatically create a folder with the same name as the ZIP file (without the ".zip" extension) in the same directory. In this case, it will create a folder called "CipherChat". (If it is not called CipherChat rename it)

Or use the following command in the folder where the file CipherChat.zip or CipherChat-master.zip is located:
unzip CipherChat.zip -d CipherChat
or
unzip CipherChat-master.zip -d CipherChat

Use the following command in the folder where the file CipherChat.zip or CipherChat-master.zip is located:
unzip CipherChat.zip -d CipherChat
or
unzip CipherChat-master.zip -d CipherChat

6. Install all required packages:

If you have not yet installed Python, install it at https://www.python.org/downloads/ by clicking on Download Python and install it.

Use cd CipherChat to get into the right folder.

Use the following command to install all packages:
python3 -m pip install -r requirements.txt

Or create a virtual environment with python3-venv and install the packages:
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt

7. Done!

You have successfully downloaded, verified, and unzipped CipherChat. Use python main.py -h or .venv/bin/python main.py -h to get an overview of all commands.

8. Use of this hidden service:

If you now start the client with python main.py, enter this hidden service as your CipherChat chat server: {{ hidden_service_hostname }}
